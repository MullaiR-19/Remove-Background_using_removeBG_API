# Image Background Remover using remove.bg API

Removing backgrounds from images is a common task in various domains such as graphic design, e-commerce, and content creation. This project provides a simple and efficient solution to automatically remove backgrounds from images using the remove.bg API.

## Introduction

The **Image Background Remover** project leverages the remove.bg API, which utilizes advanced AI algorithms to accurately detect and remove backgrounds from images. This tool saves users significant time and effort compared to manual background removal, making it perfect for professionals and enthusiasts alike.

## Features

- Seamless integration with the remove.bg API.
- User-friendly Tkinter GUI
- Option to process single images at a time.
- Save to output file to the same path of the orginal file

## Installation

1. Clone the repository: `git clone https://github.com/yourusername/image-background-remover.git`
2. Navigate to the project directory: `cd image-background-remover`
3. Install dependencies: `pip install -r requirements.txt`

## Usage

1. Obtain your remove.bg API key by signing up at [remove.bg](https://www.remove.bg/).
2. Replace `YOUR_API_KEY` with your actual API key.
